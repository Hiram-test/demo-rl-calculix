# Demo RL: Abaqus and Gmsh + CalculiX

This public repository publishes two archived versions of the demo reinforcement-learning workflow:

1. `demo.zip` — the original Abaqus-based demo package.
2. `gmsh_calculix_rl_package.zip` — the migrated reinforcement-learning package using the Gmsh + CalculiX toolchain.

The archives are distributed as assets of the [`v1.0.0` release](../../releases/tag/v1.0.0), rather than as ordinary Git objects.
