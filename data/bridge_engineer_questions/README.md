## 数据边界

- 这是公开问题样本，不是统计抽样调查；
- 一个帖子可拆成多个原子问题；
- 问题为人工转述，不复制来源正文；
- 论坛回答不作为权威技术结论；
- 英语软件社区来源占比较高；
- 频次只能解释为“在本样本中反复出现”。
