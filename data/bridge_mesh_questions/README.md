# 桥梁有限元网格问题语料

本目录只记录**网格本身的工程问题**。它不再把一般建模、材料、接触或验证问题混入统计；只有当问题直接要求作出网格决定时才收录。

## 当前规模

- 36 个唯一公开来源页面；
- 103 个经人工转述和拆分的原子网格问题；
- 29 个桥梁专属来源页面，对应 81 个问题；
- 7 个可直接迁移到桥梁结构的通用结构网格来源页面，对应 22 个问题。

## 文件

- `mesh_only_questions_01.csv`—`mesh_only_questions_04.csv`：逐条问题；
- `summary.json`：来源、渠道和网格需求族统计；
- `../../docs/bridge_fea_mesh/MESH_ONLY_QUESTION_FINDINGS.md`：第一轮发现和产品含义。

四个 CSV 使用同一字段：

| 字段 | 含义 |
|---|---|
| `question_id` | 原子网格问题 ID |
| `source_group_id` | 原始帖子或 FAQ 的来源组 |
| `source_scope` | `bridge_specific` 或 `structural_mesh_transfer` |
| `channel` | 来源社区或官方支持站 |
| `year` | 页面发布时间或问题时间 |
| `component_or_context` | 桥梁构件或可迁移结构场景 |
| `mesh_need_group` | 网格需求族 |
| `question_paraphrase` | 对原问题的中文人工转述 |
| `decision_needed` | 工程师真正需要作出的网格决定 |
| `qoi_or_result` | 受网格决定影响的结果量 |
| `source_title` | 原页面题名 |
| `url` | 可追溯来源 URL |

## 收录边界

- 问题必须直接涉及尺寸、密度、阶次、形状、过渡、连通、局部细化、子模型、奇异性、网格收敛、结果恢复或重划分；
- 只问材料参数、边界条件、接触公式或求解器参数的问题不收录；
- 当网格与接触、连接或结果恢复不可分时，只编码其中需要作出的网格决定；
- 问题文本为人工转述，不复制长段原文；
- 论坛答案不作为技术权威，技术规则仍需由标准、官方理论、验证例或正式研究确认。

## 统计边界

这不是概率抽样调查。英语软件社区和遇到困难后公开提问的用户被过度代表；一个来源页面也可能拆成多个原子问题。因此频次只能解释为“在本次公开网格问题样本中反复出现”，不能解释为桥梁工程师总体比例。
