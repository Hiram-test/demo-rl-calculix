# MeshPilot TQZ 真实热点与坐标收尾 PSO：真实 VM 结果

## 运行边界

本结果来自 GitHub Actions Ubuntu 24.04、真实 Gmsh 和真实 CalculiX。六个图纸参数化 TQZ(XII) 算例分别运行 reference、uniform coarse 和四种优化方法。

没有使用代理模型、假求解器、降级模拟、手工目标值或经验公式替代有限元。下文“FE 次数”指优化阶段新增的真实唯一 CalculiX 调用；每个优化运行均受 32 次硬上限约束。

该模型是局部 C50 三维线弹性算法基准，不是支座、垫石或整孔箱梁的工程验收计算。

## 四种方法

| 标识 | 候选区 | 优化算法 | 迁移 |
|---|---|---|---|
| `fixed_round_cold` | 固定加载足迹 3×2 | 原连续位置取整 PSO | 无 |
| `hotspot_round_cold` | 当前算例真实 coarse 应力热点 | 原连续位置取整 PSO | 无 |
| `hotspot_polished_pso_cold` | 与上一行相同热点 | PSO 全局搜索＋有限坐标收尾 | 无 |
| `hotspot_polished_pso_transfer` | 与上一行相同热点 | 同一坐标收尾 PSO | 归一化热点匹配＋严格真实门卫 |

## 三层收益被单独量出来

### 1. 候选区层：固定区域 → 真实 coarse 热点

保持原 PSO 不变，仅替换候选区：

- FE 次数：192 → 192；
- 平均目标值改善 4.55%；
- 平均相对位移误差降低 66.86%。

因此热点收益不是 PSO 公式造成的。候选区来自每个构件自己的真实 coarse CalculiX 响应。

### 2. 算法层：原 PSO → 坐标收尾 PSO

保持热点完全相同，不使用迁移：

- FE 次数：192 → 187，减少 2.60%；
- 平均目标值：0.004171013 → 0.003841452，改善 7.90%；
- 平均相对误差：0.088745% → 0.078080%，降低 12.02%；
- 6/6 保持可行。

坐标收尾利用“每个热点只有 L0–L3 四个等级”的结构，把 PSO 后半段预算用于逐维检查尚未计算的等级，而不是继续产生大量相近或重复粒子。

### 3. 业务层：冷启动 → 严格热点迁移

保持热点和坐标收尾算法不变，仅加入批量历史：

- FE 次数：187 → 178，减少 4.81%；
- 平均目标值：0.003841452 → 0.003595181，改善 6.41%；
- 平均相对误差：0.078080% → 0.075629%，降低 3.14%；
- 6/6 保持可行。

迁移门卫要求 warm probe 在当前构件上真实运行，并且目标值不高于 coarse 的 75%。五次存在历史来源的机会中，2 次接受缩减预算，3 次恢复完整预算。

## 最终方法相对原固定区域基线

- 真实优化 FE：192 → 178，减少 **7.29%**；
- 平均目标值改善 **17.73%**；
- 平均相对位移误差降低 **71.76%**；
- 平均单元数由 1218.3 增至 2044.0，因此相对固定区域基线，平均优化墙钟时间增加 **22.61%**；
- 但相对使用相同热点的原 PSO，最终方法的平均优化墙钟时间由 6.729 s 降至 5.537 s，减少 **17.71%**；
- 可行性保持 **6/6**。

## 为什么门卫从早期规则收紧

早期门卫只要求 warm probe 不比 coarse 差超过 20%。真实 VM 发现该规则会接受“虽然优于 coarse，但不足以支撑缩预算”的历史解，导致迁移平均误差上升。

最终规则改为：

\[
J_{warm}\le 0.75J_{coarse}
\]

也就是历史解至少让当前构件的 coarse 目标值改善 25%，才允许减少 FE 预算。收紧后，迁移同时实现 FE 减少、目标改善和误差改善。

## 被淘汰的算法版本

纯“每轮只能 ±1”的阶梯式 PSO，以及它的窄群体和坐标收尾版本，都在相同真实 FE 上限下完成过六算例 VM 对照，但平均质量不如热点上的原 PSO，因此没有被包装成成功结论。

最终提升为主方法的是：

> **真实 coarse 热点＋PSO 全局搜索＋有限坐标收尾＋严格热点匹配迁移。**

## 不能推出的结论

本实验不能证明：

- 结果可直接用于支座、垫石或箱梁验收；
- 接触、钢筋、预应力或开裂效应已被覆盖；
- 所有构件族都能稳定节省 7.29% FE；
- 六个构件已经足够确定大批量迁移收益。

后续需在更密集的同族参数序列和真实三维构件族上重复同样的 A→B→C→D 分层实验。
