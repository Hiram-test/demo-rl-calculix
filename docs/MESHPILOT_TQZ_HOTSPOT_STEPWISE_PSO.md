# MeshPilot：真实热点候选区、坐标收尾 PSO 与批量迁移

## 主方法已经从固定区域改为真实粗网格热点

旧 TQZ 基线把支座加载足迹机械划成 3×2 六块。它只保留为 A 组对照，不再代表主方法。

现在每个构件都独立执行一次真实的均匀粗网格分析：

1. Gmsh 生成三维 C3D4 网格；
2. CalculiX 完成线弹性静力求解；
3. 根据真实 CalculiX 节点位移重构每个 C3D4 单元的等效应力；
4. 将局部块体上部划为 5×4×3 个虚拟 cell；
5. cell 分数由应力水平和与相邻 cell 的应力差共同组成；
6. 相邻高分 cell 合并；
7. 选择前 6 个区域作为当前算例的网格优化候选区。

候选区来自当前构件自己的粗网格结果。一经选定，在该算例的一次 PSO 中保持固定；粒子只决定六个热点分别采用 L0、L1、L2 或 L3。

整个流程不使用代理模型、假求解器、响应面或降级模拟。reference、coarse、warm probe 和每个优化候选均由真实 Gmsh/CalculiX 计算。

## 算法层：PSO 全局搜索后做有限坐标收尾

直接把连续 PSO 四舍五入虽然能搜索，但最后一部分预算容易被重复粒子或相近粒子消耗。这里利用一个简单结构：

> 每个热点只有四个网格等级。找到一个较好组合后，沿一个热点维度把其余等级看完，最多只需再看三个位置。

接受的方法为 `coordinate_polished_rounded_pso`：

1. 保留 PSO 的全局群体搜索；
2. 将粗网格热点分数最高的两个候选初始化为 L2，中间两个为 L1，其余为 L0，作为一个确定性引导粒子；
3. 在 32 次真实唯一 FE 硬预算中，先给 PSO 群体分配 14 次左右的全局搜索预算；
4. 剩余预算按热点分数从高到低做坐标收尾；
5. 固定其余五维，把当前维的 L0–L3 中尚未计算的位置逐一真实求解；
6. 若某一等级更好，立即更新当前最好解，再处理下一个热点；
7. 重复离散位置仍在调用求解器之前补到未评价位置。

这不是局部代理或解析近似，而是对四级离散结构的真实有限元枚举收尾。

曾经实现并真实测试过“每轮只能 ±1 的纯阶梯 PSO”。它在同样的 32 次 FE 预算下质量明显较差，因此没有被提升为主方法；代码和实验仅保留为算法淘汰记录。最终采用的是“PSO 全局搜索＋有限坐标收尾”。

## 业务层：按热点物理位置迁移，而不是复制编号

历史构件为每个热点保存：

- 相对局部块体的归一化中心；
- 归一化长、宽、高；
- 粗网格热点分数；
- 最优网格等级。

当前构件完成自己的 coarse 热点检测后，通过中心、尺寸和分数进行一对一匹配。旧热点的等级只传给物理位置匹配的新热点，不再使用“旧 patch 1 直接复制给新 patch 1”的方式。

迁移后必须执行一次真实 CalculiX warm probe。最终门卫采用更严格的业务规则：

- warm probe 必须可行；
- warm probe 目标值必须不高于当前 coarse 目标值的 75%，也就是至少改善 25%；
- 只有满足两项，才按构件相似度缩减真实 FE 预算；
- 否则拒绝迁移，并恢复完整的 32 次唯一 FE 预算。

这一阈值不是为了制造更大的 FE 降幅。早期 20% 宽松门卫会接受一个改善不够充分的历史解，虽然省计算，但平均精度退化。真实 VM 对照后，最终收紧为“至少改善 coarse 25%”。

## 四组真实对照

| 组别 | 候选区域 | 优化算法 | 批量迁移 | 作用 |
|---|---|---|---|---|
| A | 固定加载足迹 3×2 | 原连续位置取整 PSO | 无 | 原基线 |
| B | 真实 coarse 应力热点 | 原连续位置取整 PSO | 无 | 单独验证热点选择 |
| C | 与 B 完全相同的热点 | PSO＋有限坐标收尾 | 无 | 单独验证 PSO 算法改进 |
| D | 与 B 完全相同的热点 | 与 C 相同 | 热点匹配＋严格真实门卫 | 单独验证批量业务收益 |

每个优化运行最多新增 32 次唯一 CalculiX 求解。四组使用相同材料、荷载、几何、reference、目标函数和单元预算。

## 已验证的分层收益

六个真实 TQZ 算例全部完成，四种方法均为 6/6 可行：

- A→B，单独使用真实热点：FE 次数不变，平均目标值改善 4.55%，平均相对位移误差降低 66.86%；
- B→C，单独加入坐标收尾 PSO：真实优化 FE 从 192 降到 187，平均目标值再改善 7.90%，平均误差再降低 12.02%；
- C→D，单独加入严格批量迁移：真实优化 FE 从 187 降到 178，平均目标值再改善 6.41%，平均误差再降低 3.14%。

最终 D 相对固定区域 A：

- 真实优化 FE：192 → 178，减少 7.29%；
- 平均目标值：0.004369958 → 0.003595181，改善 17.73%；
- 平均相对位移误差：0.267783% → 0.075629%，降低 71.76%；
- 可行性：6/6 → 6/6。

五次存在历史来源的机会中，严格门卫接受 2 次、拒绝 3 次。拒绝的算例恢复完整预算，没有为了追求调用次数而强行迁移。

## 解释边界

模型仍是支座附近的局部 C50 三维线弹性块体，用于研究网格优化算法。接触、摩擦、钢筋、预应力、混凝土开裂和整孔箱梁全局响应未包含。所有结果仍需工程人员复核，不能作为支座、垫石或梁体的工程验收结论。
